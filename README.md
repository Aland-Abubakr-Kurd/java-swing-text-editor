# Java Swing Text Editor

A lightweight text editor implemented in Java Swing.  
Supports typing, font customization, color changes, and opening/saving `.txt` files.

---

## • Features

- Change font style and size
- Pick custom font colors
- Open `.txt` files
- Save your text to a file
- Scrollable text area with line wrap
- Exit button to close the program

---

## • Technologies Used

- Java
- Swing (GUI components)
- AWT (Fonts, Colors, Graphics)
- File I/O (Scanner, PrintWriter)
- Event handling (ActionListener, ChangeListener)
